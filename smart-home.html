Smart Home Content

This was a project in my class to create a website demonstrating all the unique technologies that you could find in the novel 1984 written by George Orwell. I used Figma to brainstorm and develop the design for the site. My site is mostly built out of HTML for the structure, and CSS for the styling.

I really had to use time management and be more persistent when I worked because since it was one of my first completed websites I had to make sure I had enough time to figure out anything I needed help for, so I did not fall behind. I also used high quality images since there were no images of the technology in the book.
